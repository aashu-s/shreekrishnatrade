## 🚀 DEPLOY TONIGHT - 3 SIMPLE STEPS

### **STEP 1: Create Deployment Package** (2 minutes)
```bash
# Double-click this file:
create-deployment-package.bat
```
This creates a clean folder with only production files.

### **STEP 2: Configure for Production** (5 minutes)
1. Open `includes/config.php` in the package
2. Update these values:
   ```php
   define('DB_NAME', 'your_database_name');
   define('DB_USER', 'your_database_user');
   define('DB_PASS', 'your_database_password');
   define('SITE_URL', 'https://yourdomain.com');
   define('ADMIN_EMAIL', 'admin@yourdomain.com');
   ```

### **STEP 3: Upload & Deploy** (10-15 minutes)
1. **Upload Files** via FTP/cPanel
2. **Import Database** (`database/schema.sql` in phpMyAdmin)
3. **Test**: Visit your domain
4. **Login**: `yourdomain.com/admin/login.html`
   - Username: `admin`
   - Password: `Admin@123` (CHANGE THIS!)
